# Github Top Typescript Projects

Приложение для просмотра самых популярных (по звездам) TS проектов Github

## Разработка

Запуск проекта:

### `npm start`

Проект запускается на [http://localhost:3000](http://localhost:3000)

## Тестирование через Playwright

### `npm test`

## Сборка проекта

### `npm run build`
